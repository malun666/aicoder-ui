# aicoder-ui

> A Vue.js 2.0 UI Toolkit for Web.

## Links

## Install

```shell
npm install aicoder-ui -S
```

## Quick Start

``` javascript
import Vue from 'vue'
import AicoderUI from 'aicoder-ui'

Vue.use(AicoderUI)

// or
import {
  Button
  // ...
} from 'aicoder-ui'

Vue.component(Button.name, Button)
```
